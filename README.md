# Median and MAD

##### Description

The `Median and MAD operator` computes the median and Median Absolute Deviation (MAD) of the data points in a cell.

##### Usage

Input projection|.
---|---
`y-axis`        | numeric, measurement 

Output relations|.
---|---
`median`    | median, per cell
`mad`        | median absolute deviation, per cell
